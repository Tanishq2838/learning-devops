# Day 1 - Linux Filesystem & Navigation

## Topics Covered
- Linux filesystem hierarchy (/home, /etc, /var, /tmp, /bin, /usr)
- Navigation commands (cd, pwd, ls)
- File operations (touch, mkdir, cp, mv, rm)
- Reading files (cat, tac, head, tail)
- Writing to files (echo, cat >, cat >>)
- Absolute vs relative paths
- Git basics - init, add, commit, push
- Resolved first Git merge conflict

## Files
- linuxnotes.txt - notes taken during learning
- commands.txt - all commands learned today with descriptions
- questions.txt - doubts and questions

## Key Takeaways
- Absolute path always starts with /
- >> appends, > overwrites
- rm has no undo, always double check
- Git conflict happens when local and remote are out of sync
